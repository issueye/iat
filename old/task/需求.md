# `iat` 工具

`iat` 是一个 `ai` 工具，类似与 `claude code`，使用了 `golang` 编写，使用了 `wails v2` 框架，实现了 `ai` 工具的功能。

## 技术栈

    - 后端
        - `golang`
        - `wails v2`
        - `goja` (github.com/dop251/goja)
        - `sqlite` (github.com/glebarez/sqlite)
        - `gorm` (gorm.io/gorm)
        - `mux` (github.com/gorilla/mux)
        - `slog` (使用原生日志)
        - `eino` (github.com/cloudwego/eino) 字节跳动 AI 框架

    - 前端
        - `vue3`
        - `vue-router`
        - `pinia`
        - `pinia-plugin-persistedstate`
        - `vite`
        - `navie ui` (github.com/navie-ui/navie)
        - `javascript`
        - `axios`

## 主要技术点

- 通过 `golang goja` 实现动态脚本
- 通过动态脚本自定义 `agent`
- 前端项目是依赖与 `wails v2` 框架，使用了 `vue3` 编写，使用了 `navie ui` 组件库。
- 使用 `sse` 实现 ai 与前端的通信。
- 前端项目是单页应用，使用了 `vue-router` 实现路由，使用了 `pinia` 实现状态管理，使用了 `pinia-plugin-persistedstate` 实现状态持久化。
- 前端项目使用了 `javascript` 编写，使用了 `vue3` 提供的 `setup` 语法糖。
- 前端项目使用了 `axios` 实现 http 请求。
- 前端项目使用了 `vue3` 提供的 `onMounted` 钩子函数，在组件挂载时发送 http 请求，获取 `agent`、`script`、`tool` 列表。

## 功能

- `agent` 管理模块

  - 可以添加、删除、修改 `agent`
  - 可以查看所有 `agent`

- `script` 管理模块

  - 可以添加、删除、修改 `script`
  - 可以查看所有 `script`

- `tool` 管理模块

  - 可以添加、删除、修改 `tool`
  - 可以查看所有 `tool`

- `MCP` 管理模块

  - 可以添加、删除、修改 `MCP`
  - 可以查看所有 `MCP`

- 会话管理模块

  - 可以添加、删除、修改会话
  - 可以查看所有会话

- 项目管理模块

  - 可以添加、删除、修改项目
  - 可以查看所有项目

- AI 模型管理模块
  - 可以添加、删除、修改 AI 模型配置（提供商、API Key、Base URL 等）
  - 可以查看所有 AI 模型
  - 支持模型连接测试
