# iat 项目构建与发布计划

## 项目状态回顾
我们已经完成了 `iat` 工具的所有核心功能开发（Phase 1 - Phase 5），包括：
1.  **基础架构**: Wails + Vue3 + SQLite，前后端通信与数据库层。
2.  **核心引擎**: 
    - `goja` 脚本引擎（支持自定义脚本执行）。
    - `eino` AI 引擎（支持 OpenAI/DeepSeek 等模型接入，SSE 流式响应）。
3.  **业务模块**:
    - **项目管理**: 项目的增删改查。
    - **AI 模型管理**: 模型配置与连通性测试。
    - **脚本管理**: 在线编写、测试和管理 JS 脚本。
    - **Agent 管理**: 定义 Agent 的角色和模型配置。
    - **会话管理**: 完整的聊天界面，支持 Agent 选择、Markdown 渲染、代码高亮和流式对话。
